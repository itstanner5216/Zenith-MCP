# Legacy tests — old (pipeline-era) zenith-toon architecture

These tests target the **deleted** pipeline-era architecture (`string-codec` /
`pipeline` / `dedup` / `encoder` / `router` / `config` / `presets`). They are
retained for reference only, pending a rewrite against the refactored engine-chain
(`compress-source` → SageRank → BMX+ → removal → render). They are **not** expected
to compile or pass, and are not wired into the build. Relocated during the Phase 2
toon-refactor migration.
